# FinSight AI

AI-powered personal finance intelligence platform that helps users analyze spending patterns, forecast future expenses, improve budgeting decisions, and gain actionable financial insights.

## Features

- Expense Tracking
- Spending Dashboard
- Category Analytics
- AI Categorization
- Spending Forecasting
- Budget Recommendations

## Tech Stack

- React / Next.js
- Laravel
- MySQL
- Python
- Scikit-Learn
